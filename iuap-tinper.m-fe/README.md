# TinperM - BIP 移动端基础组件库

## 特性

- 体积小，高性能
- 基于用友 `YonDesign` 移动端设计体系
- 支持 `Design Token` 精细化、智能化主题定制能力
- 国际化特性，支持多语言、多时区、多格式
- 移动端微动效、手势优化
- 组件示例丰富
- API文档、使用指导文档齐全
