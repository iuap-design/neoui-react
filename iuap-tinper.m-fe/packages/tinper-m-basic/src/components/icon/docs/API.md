## Icon 属性
### API

属性 | 说明 | 类型 | 默认值
----|-----|------|------
type | 类型 | `string` | -
color | 颜色 | `string` | -
size | 尺寸 | `'sm' \| 'md' \| 'lg'` | `'md'`
className | 样式类名 | `string` | -
style | 自定义样式 | `React.CSSProperties` | -
fieldid | dom标识 | `string` | -
clsPrefix | class前缀 | `string` | `'mui'`
onClick  | 点击事件 | `(event: React.MouseEvent<HTMLButtonElement, MouseEvent>) => void` | -

| --------------- | ---------------- |
