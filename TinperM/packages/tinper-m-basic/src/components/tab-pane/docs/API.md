## API

属性 | 说明 | 类型 | 默认值
----|-----|------|------
children | 子项 | `any` | -
key | `key` | `string` | -
label | 标签 | `string` | -
className | 样式`class` | `string` | -
style | 样式`style` | `React.CSSProperties` | -
minHeight | 最小高度 | `string` | -
width | 宽度 | `string` | -
bgColor | 背景颜色 | `string` | -
marginBottom | 下边距 | `string` | -
